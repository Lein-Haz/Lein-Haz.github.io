Summary

I'm a fullstack developer that has a few years of work experience now. In that time I've learned a lot, but there is still lots of knowledge out there to acquire. Luckily, as someone that is an eager learner and tinkerer, I shouldn't be getting bored anytime soon.

I was born in Ulm, Germany, but was in Ravensburg for much of my early youth. We moved to St. Louis, MO as I was entering grade school, where I spent my adolescence when I wasn't visiting Germany. I'd like to believe those experiences made me a more open person.

My first foray into programming was with Java and C++, a lot of console apps as you can imagine. Early on I stayed away from front end work, though I did play around with Java Swing a bit which looks really dated now. I guess I also had some biases toward front end development. I had this misconception of front end work being a lot fiddling with static html, a remnant of early web days.

While still in school I got really interested in mobile development, programming for a computer in your pocket seemed like a nifty idea. Other than a really terrible HTML page I did for another class, this really exposed me to front end design concepts. A guy in the class showed me something called "quantum paper", something that would evolve into material design.

After that class I got an opportunity to do more web development. This let me work on my front end skills, allowing me to become adept at layout frameworks like Bootstrap and Flex. This changed my understanding of what modern web development is, now that application logic could be put on the client side.
