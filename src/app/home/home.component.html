So why is this site here? Well, not only to showcase my skills, but a few other reasons. I've never really built a site for myself that I put online, so that was part of it. I've also been wanting to play around with Oauth, which was fun.

Though the primary thing I wanted to play around with is Progressive Web Apps and this site is a PWA. So go visit some pages let the service worker cache things up and try coming back when you are offline.

*Disclaimer* The page sections with google maps won't work when you are offline, as you do need an active connection to load map data.
